HTML:

Acrônimo para Hypertext Markup Language, o HTML é uma Linguagem de Marcação utilizada para a construção de websites.
Utiliza de tags e através desses comandos, conseguimos estruturar o que queremos que o navegador compreenda e traduza para o leitor em um formato visual.
É desta forma que os sites são construídos primariamente.
Para quem quer saber mais, coloquei uns links úteis aí embaixo. Boa navegação, marujo!
